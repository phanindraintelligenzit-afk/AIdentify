# AIdentify Mission Control Dashboard

FastAPI + Jinja2 dashboard for the AIdentify agent swarm.

## Features
- Agent status monitoring (OWL + 4 specialists)
- Knowledge base status
- Pipeline status
- Cron job monitoring
- Marketplace projects
- Quick actions

## Run
```bash
cd /c/Users/Admin/Projects/AIdentify-marketplace/dashboard
pip install -r requirements.txt
python main.py
```
Then open http://localhost:8080
